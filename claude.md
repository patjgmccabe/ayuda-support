# Ayuda Support Website

Project Goal:
Build a modern, clean, professional website for Ayuda Support.

Core Pages:
1. Home
2. Support Broker Services
3. OPWDD Questions
4. Contact Us

Home Page Requirements:
- Hero section with welcoming headline
- Short mission statement
- Three large service cards linking to:
  - Support Broker Services
  - OPWDD Questions
  - Contact Us
- Clean layout, soft colors, accessible typography
- Mobile responsive

Navigation:
- Top navigation bar with links to all pages
- A language toggle button at the top right
- The button switches all visible content between English and Spanish
- The language toggle must work across all pages
- Use a simple JavaScript-based translation system (no external APIs)
- Store translations inside a structured object
- Default language: English

Support Broker Services Page:
- Clear explanation of services
- Bullet list of what is included
- Professional and reassuring tone

OPWDD Questions Page:
- FAQ-style layout
- Expandable/collapsible question sections
- Clear and simple explanations

Contact Us Page:
- Simple contact form (Name, Email, Message)
- Professional layout
- No backend required yet (frontend only)

Design Requirements:
- Modern design
- Plenty of white space
- Rounded corners
- Soft shadows
- Calm and welcoming feel
- Accessibility-conscious
- Responsive design

Technical Requirements:
- Use plain HTML, CSS, and JavaScript
- No heavy frameworks
- Use clean folder structure
- Keep code readable and organized
